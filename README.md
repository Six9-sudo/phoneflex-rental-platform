# 📱 PhoneFlex – Premium Phone Rental Platform

A modern web platform that allows users to rent high-end smartphones like iPhones and flagship Android devices on a monthly basis instead of purchasing them outright.

---

## 🌐 Live Demo
👉 https://phoneflex-2500sv.netlify.app/

---

## 💡 Problem

Smartphones today cost ₹50,000–₹1,50,000, making it difficult for many users to afford premium devices.

---

## ✅ Solution

PhoneFlex enables users to:
- Rent premium smartphones monthly  
- Avoid high upfront costs  
- Upgrade devices anytime  
- Return devices flexibly  

---

## ⚙️ Features

- 🎨 Premium Apple-style UI/UX  
- 📱 Device filtering (iPhone / Android)  
- 🛒 Checkout flow (UI-based modal)  
- 💬 WhatsApp integration for instant leads  
- ⚡ Smooth animations and transitions  
- 📲 Fully responsive design  

---

## 🧠 How It Works

1. User visits the platform  
2. Browses available devices  
3. Selects a phone  
4. Proceeds to checkout  
5. Gets redirected to WhatsApp for confirmation  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (modern UI, gradients, animations)  
- JavaScript (DOM manipulation, filtering, checkout logic)
